#MaxVote
This web application is my recreation of https://votingapp.herokuapp.com/, while following
the FreeCodeCamp curriculum. It is a multi-page AngularJS-based web application.

###Tools
  * AngularJS
  * NodeJS
  * ExpressJS
  * SASS
  * Jade
  * MongoDB

##Change Log
